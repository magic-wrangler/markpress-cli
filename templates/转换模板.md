**表头横向合并**（左侧单元格留空，或写 `<<`）：

| 合并表头 | << |
| :---- | :---- |
| 左列 | 右列 |

**纵向合并**（下方单元格留空，或写 `^^`）：

| 类别 | 内容 |
| :---- | :---- |
| A 类 | 第一行 |
| ^^ | 第二行（向上合并到「A 类」） |
| B 类 | 第三行 |

> 表格合并需在主题 JSON 中开启 `enableRowMerge` / `enableColMerge`（内置主题默认已开启）。

# 2. 附录

## 2.1 转换方式

| 方式 | 命令 |
| :---- | :---- |
| 交互转换 | 在文档目录执行 `mpr`，选择主题与文件 |
| 命令行 | `mpr convert --md 转换模板.md --theme 有底色 --out output/转换模板.html` |
| 自定义主题 | `mpr convert --md 转换模板.md --theme ./我的主题.json` |

## 2.2 内置主题

- **有底色**：表头深蓝，适合正式文档、规则协议
- **无底色**：简洁白底

执行 `mpr themes list` 查看详情。
